# Layout One Project – Migración a React

Este proyecto corresponde a la migración de una maquetación desarrollada originalmente en **HTML y JavaScript puro** a una aplicación construida con **React**.

El objetivo principal es aplicar conceptos de **componentización**, **enrutamiento** y **arquitectura base** vistos en clase, manteniendo la fidelidad visual del diseño original.

---

## 🛠️ Tecnologías utilizadas

- React
- Vite
- React Router DOM
- JavaScript (ES6+)
- CSS

---

## 📂 Arquitectura del proyecto

El proyecto está organizado siguiendo una estructura clara y modular:

src/
├── components/
├── pages/
├── routes/
├── layouts/
├── styles/
├── App.jsx
└── main.jsx

Esta estructura permite una mejor separación de responsabilidades y reutilización de componentes.

---

## 🚀 Funcionalidades principales

- Navegación entre páginas usando **react-router-dom**
- Componentización de la interfaz (Navbar, Footer, Hero, Cards, etc.)
- Layout principal reutilizable para la aplicación
- Formulario funcional que captura datos por teclado y los procesa desde React
- Fidelidad visual respecto al diseño original en HTML/JS

---

## ✅ Requisitos cumplidos

- Uso de React
- Enrutamiento con react-router-dom
- Mínimo 5 componentes bien definidos
- Arquitectura base organizada
- Convenciones de nombres en React
- Historial de commits que evidencia el proceso de desarrollo
- Formulario funcional con captura de datos
